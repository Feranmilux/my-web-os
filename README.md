# 🚀 RetroSpace OS - Web Operating System

Welcome to **RetroSpace OS**, a fully interactive, retro cyberpunk-themed operating system designed to run completely inside your web browser. Built from scratch using raw HTML, CSS, and vanilla JavaScript.

This project was built for the **WebOS 1 Mission** in Hack Club Arcade!

## 🌟 Live Demo
👉 **[CLICK HERE TO RUN THE OS](https://feranmilux.github.io/my-web-os/)**

---

## 🛠️ Features
*   **Multi-Window Desktop:** Clean, responsive workspace environment.
*   **Draggable Interface:** Smooth desktop windows powered by precise JavaScript mouse event tracking.
*   **No Passwords:** Open-source and accessible instantly for anyone to test out.
*   **Custom Feature (Synth Pad):** A unique app built using the native Web Audio API that generates standalone audio synthesizers upon clicking user interface triggers.
